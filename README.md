# Incident-Hub
Plataforma inteligente de gestión de incidencias y accidentes en la República Dominicana.

## Configuración del entorno de desarrollo

### Requisitos
- Node.js (v18+)
- npm
- Git

### Instalación
1. Clonar el repositorio
2. Ejecutar `npm install` en la carpeta backend
3. Ejecutar `node src/index.js`
4. Abrir http://localhost:3000
